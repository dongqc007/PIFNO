# PIFNO

This repository gives the codes for ["Toward the Fast Electromagnetic Solver with Physics-Informed Fourier Neural Operator"](https://www.techrxiv.org/users/1007622/articles/1370678-toward-the-fast-electromagnetic-solver-with-physics-informed-fourier-neural-operator). The related paper has been submitted to IEEE TAP. We will release the codes after acceptance or as required by the journal.
